# YOUTUBE VIDEO/AUDIO DOWNLOADER

Description:
A python project to download audio/video from youtube.

How to Use:

1. clone the repo.
2. install pytube -> (pip install pytube)
3. run index.py
4. enter the url of the video which you want to download.
